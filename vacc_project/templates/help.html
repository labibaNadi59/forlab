Welcome to the Help Page of ShishuShield.BD. Here are some common instructions for using the system.


👨‍👩‍👧 For Parents

  • Register and login to your account.
  • Add your child’s information.
  • Book vaccination appointments.
  • Reschedule appointments before hospital acceptance.
  • View reminders and appointment status.
  • Rate hospitals after vaccination completion.

🏥 For Hospitals

  • Register your hospital account.
  • Wait for admin approval.
  • View and manage appointments.
  • Update appointment status.

🧑‍💼 For Admin

  • Approve or reject hospitals.
  • Monitor appointment records.
  • View child and parent data.

If you face any issue while using the system, please contact the system administrator.
